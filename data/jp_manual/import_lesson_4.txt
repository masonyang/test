【あります、います表示事物的存在】
```
あります表示花、草、树木、桌子事物的存在。【熟悉】

* 语序和中文一样。

[句型1] 场所 　に（有）　物 　　が　あります
	　　へや　に　つくえ　が　　あります
	（房间里有桌子。）

[句型2]　物 　は（在）　场所 に　あります
	　　いす　は　へや　に　あります
	（椅子在房间里。）
```
```
います表示人、动物、昆虫事物的存在.【熟悉】

[句型1] 场所 に　人 　　が　います
	  へや　　に　ねこ　が　います
	  （房间里有一只猫。）

[句型2]　人 　　　は　场所 　に　います
	 よしださん　　は　にわ　に　います
	（ 吉田先生在院子里。 ）
```
```
小野先生的家在哪里？【了解】
おのさんのいえ　は　どこ　ですか。
おのさんのいえ　は　どこに　ありますか。

林先生在哪里？【了解】
はやしさん　どこ　ですか。
はやしさん　どこに　いますか。

```

【ど　名词 ‘和’ 名词：表并列】
```
とけい　ど　めかね。【熟悉】
  表   和   眼镜
```

综合性的
```
いま　　に　テレビ　ど　ビデオ　が　あります。【了解】
起居室　有　电视机　和　录像机 
```

名词 の　上/下/前/后/旁边/里边/外边	
```
Page 34
うえ・した・まえ・うしろ・となり・なか・そと。【了解】
```

当说话人就某事征求听话人的同意时，句尾加助词　ね

```
えきのまえ　に　ぎんこう　あります　ね
车站前面有家银行吧。
```

疑问词 も　动词（否定）
```
	きようしつ　に　だれ　も　いません。（动词否定）
	教室里谁也没有

	れいぞうこ　に　なに　も　ありません。
	冰箱里什么也没有
```
