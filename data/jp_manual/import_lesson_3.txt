【ここ、そこ、あそこ是指示场所的词】
```
ここはデバード　です。【了解】
（这里是百货商店。）
```
```
そこは図書館　です。【了解】
（那里是图书馆。）
```
```
あそこは 入り口 です。【了解】
（那儿是入口。）
```

【表示“名词”存在于某个场所。】
```
トイレは　ここです。【了解】
（厕所在这里。）
```
```
食堂はデバードの　ななかい です。【了解】
(食堂在百货商店的7楼。)
```

【询问场所在什么地方。】
```
トイレは どこですか。【了解】
（厕所在哪里？）

	———— あちらです/あそこです。（在那儿。）【了解】

あなたのかばんは　どこですか。【了解】
	———— わたしのかばんは　ここです。【了解】
```

【表示'也'】

	あなたはにほんじんです。【了解】

	わたじもにほんじんです。【了解】
